PPID Polrestabes Makassar - Static Website (Local demo)
------------------------------------------------------

Isi folder:
- index.html (home + upload form)
- berita.html (daftar berita + search + filter)
- detail.html (halaman detail berita)
- dashboard.html (admin: tambah/edit/hapus berita)
- about.html, agenda.html, regulasi.html, layanan-informasi.html, standar-layanan.html, contact.html (static pages)
- assets/css/style.css
- assets/js/news.js
- assets/js/dashboard.js
- assets/img/ (letakkan logo.png dan hero.jpg jika ada)

Cara pakai:
1. Buka index.html di browser (double click).
2. Gunakan form di halaman Home untuk menambah berita (gambar akan disimpan sebagai base64 di localStorage).
3. Buka berita.html untuk melihat daftar berita. Klik judul untuk buka detail.html?id=<index>.
4. Buka dashboard.html untuk edit dan hapus berita (operasi lokal di localStorage).
